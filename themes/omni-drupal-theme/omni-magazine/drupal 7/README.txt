------------------------------------------------------------------	

To setup Omni Drupal theme, user must read & complete user guide - Omni Basic Setup first.
To setup Omni Sample Content, user should follow instruction of Sample Content Setup guide.

WARNING: Omni Sample Content Setup only suitable for newly installed Drupal site. It shouldn't install on existing site as it will delete all existing data and cannot be undone.

------------------------------------------------------------------	

Theme Folder Structure

- document
  - sample content: import files for sample content
  - user guide
  - views: import files for view backup only

- licensing

- modules: all required modules

- omni
  - omni_magazine: the actual theme folder
