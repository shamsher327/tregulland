Omni Basic Setup

- Introduction

In this section, we will show you some basic setup steps that Omni theme need. By using this guide, we assume you have basic knowledge of Drupal and have Drupal 8.x installed on your system.

- Theme Pack Folder Structure

  • theme: theme files folder
    • omni_portfolio: this is the actual theme, please upload this folder to into drupal 8 root's themes/custom folder.
  • licensing
  • modules: all required modules for Omni Drupal theme.
  • document
    • user guide
    • views: importable view files
    • sample content: demo SQL files & demo image files
  • README.txt

- Installation

There is a modules folder within the download package which include all necessary modules for this theme. Only package included version of modules were tested for compatibility. If new version released please backup before upgrade since it might not be compatible.

Please install Omni Portfolio theme and following Modules (If you don't know how to install Drupal theme, please check our Drupal Quick Start guide):

  • Admin Toolbar 8.x-1.15
  • Configuration Update Manager 8.x-1.1
  • Contact Block 8.x-1.1
  • Features 8.x-3.0-beta3
  • Field Group 8.x-1.0-rc4
  • Inline Entity Form 8.x-1.0-alpha6
  • Share Buttons by AddToAny 8.x-1.10
  • Social Media Links Block 8.x-2.1
  • Simplenews 8.x-1.0-alpha2
  • Taxonomy Views Integrator 8.x-1.0-beta1
  • Views Infinite Scroll 8.x-1.2
    Views Infinite Scroll has been modified for Portfolio theme, please use the one within download package and wait for Omni release before upgrade.
  • Omni Custom
  • Omni Feature: simply import all content type & block type for Omni theme. Available within download files.

- Basic Configuration

You need to do ALL the following steps by order, even if you think you might not need it for now.

1. Enable Omni Portfolio theme

Go to Appearance

  • Enable and set Omni Portfolio as Default theme
  • Make sure administration theme set to Seven

2. Enable required modules by order!

Go to Extend page:

  1. Enable modules Statistics, Update Manager, Admin Toolbar, Contact Block, Field Group, Inline Entity Form, Simplenews, AddToAny, Social Media Links Block, Taxonomy Views Integrator, Views Infinite Scroll then click Save Configuration
  2. Enable modules Features, Features UI, Configuration Update Base and Omni Custom then click Save Configuration
  3. Enable Omni Feature then click Save Configuration (might take longer).

3. Setup avatar

Go to Configuration > Account setting > Manage fields

  • Edit Picture field
  • Upload images/avatar.png file as default image
  • Click Save Configuration

For live site, you should enable Aggregate for JavaScript and CSS files. Now you have completed the basic setup! You can continue on Omni Feature Setup.
