Installing @font-your-face:
===========================

- Place the extracted module in your modules directory, e.g. sites/all/modules/contrib/fontyourface
- Go to Administration È Modules and enable @font-your-face and one or more of the provider submodules.
- Go to Administration È Configuration È User interface È @font-your-face settings and import fonts.

Support:
========
- See http://drupal.org/project/fontyourface#support for support options.
